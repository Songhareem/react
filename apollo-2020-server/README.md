
# GraphQL server

- $ yarn add graphql-yoga
    - graphql server api

- $ yarn add node-fetch
    - window의 Fetch API와 비슷

# GraphQL로 해결가능한 문제

- Over-fetching
    - 고객 또는 개발자가 요청 또는 필요한 영역의 정보보다 많은 정보를 서버 또는 DB로부터 받는것

- Under-fetching
    - 고객 또는 개발자의 요청을 위해 서버 또는 DB로부터 한번이 아니라 여러번 정보를 받아와야 함 

# Schema

- 사용자에게 보내거나 받을 Data에 대한 설명
