
# 프론트엔드 아카데미

## 1. Javascript & Typescript

### 1) 시작하며
- 빠르게 배워야할 것
    - 프레임워크 등
- 나중에 배워야할 것
    - 네트워크, 시스템 프로그래밍, OS 등
    - 자료구조, 알고리즘 등
- 반복해서 배워야할 것
    - 우선순위 높은것 : 나중에 배워야할 것들
    - 우선순위 낮은것 : 빠르게 배워야할 것들

- 실습준비
    - bundler 설치 (parcel)
        - npm i -g parcel-bundler
    - typescript 설치
        - npm i -g typescript or npm i --save-dev typescript (프로젝트 마다)

- 참고
    - 실무에서 자주쓰이는 번들러는 Webpack
